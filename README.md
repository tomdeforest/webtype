# WebType

A repository for exploring modern web typesetting techniques with CSS.

## Overview

This project contains examples and experiments with contemporary CSS typography features, including layout, font properties, text rendering, and responsive type systems.

## Contents

- `index.html` - Comprehensive reference of HTML text elements

## Purpose

This repository will serve as a testing ground and learning resource for:

- Advanced CSS typography properties
- Responsive typesetting techniques
- Font loading and optimization
- Text layout and composition
- Cross-browser typography rendering

## Getting Started

Open `index.html` in a web browser to view the text elements reference page.
